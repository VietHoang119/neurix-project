# Neurix MVP — Digital Brain Demo

Neurix là nền tảng "Digital Brain" giúp tổ chức, kết nối và khai thác tri thức cá nhân thành một mạng node nhẹ (quantumdata), hỗ trợ ghi nhớ, tổng hợp và tạo ra ý tưởng.
